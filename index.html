MY PROJECTS

Crumb Chronicles

A web game where players are interviewed by a sentient toaster. Originally made as text-based C++ game for a module during my first year.

Sol - Weather Dashboard

A weather application with an interactive pixel art interface. Built with vanilla JavaScript, this Progressive Web App uses the OpenWeatherMap API to display current weather conditions, forecasts, air quality data, and temperature history with custom retro-style animations. Features include geolocation, unit conversion, dark/light themes, and offline support.

CheckMate - Task Management App

A comprehensive productivity system built with vanilla JavaScript, Firebase, and external APIs. CheckMate combines task management, Pomodoro timing, and context-aware suggestions through weather integration. Features include user authentication, custom categories, dashboard analytics with visual charts, calendar views, and cloud synchronization. The app provides intelligent task prioritization suggestions based on weather conditions and personal productivity patterns.

Book Library API

A full-stack application for managing personal book collections with JWT authentication, role-based access control, and a RESTful API. Built with Node.js, Koa.js, MySQL, and React, featuring responsive design and comprehensive filtering options.

MotionUI

A gesture-controlled web interface that transforms your webcam into a touchless input device. Built with MediaPipe and TensorFlow.js, this project enables users to navigate, scroll, and click using only hand movements. Features include real-time hand tracking with a virtual cursor, smooth gesture recognition, and keyboard fallbacks.
